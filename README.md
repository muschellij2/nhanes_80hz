# nhanes_80hz

<!-- badges: start -->
<!-- badges: end -->

The goal of `nhanes_80hz` is to process the 80Hz raw data from the
[NHANES](https://wwwn.cdc.gov/nchs/nhanes/Search/DataPage.aspx?Component=Examination)
accelerometers. Namely the `PAX_H` and `PAX_G` data points.
